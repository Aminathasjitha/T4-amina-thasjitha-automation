# T4-amina-thasjitha-automation


Prerequisites
--------------

Java JDK 8 or above
Eclipse IDE (or any preferred Java IDE)
Selenium WebDriver (latest version)
TestNG library
Browser drivers (e.g., ChromeDriver, GeckoDriver)

How to Run
------------

Clone or download the project.
Open the project in Eclipse IDE.
Configure browser drivers in utils or properties file.
Run tests using TestNG XML or individual test classes.
Test reports will be generated in the test-output folder.
